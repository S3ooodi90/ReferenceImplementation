Shareable, Structured, Semantic Model

S3Model is the answer to the data modeling question for the 21st century and beyond.

Most apsects of information management have been improved and updated to accomodate our move from localized data management to a distributed infrastructure. However we have not, until now, taken this step with the core apsect of data modeling. We are still using the 1980s version of how to build a data model. S3Model fixes this problem without changing all of the good things we have accomplished such as security, SOAP, REST, authentication, authorization, etc.

Take a look at the user docs as well as the technical docs. The technical documention is available in the top six global languages.
