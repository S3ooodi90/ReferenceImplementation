Complex Type s3m:ReferenceRangeType
Namespace http://www.s3model.com/ns/s3m/
Annotations
 
Defines a named range to be associated with any ORDERED datum. Each such range is sensitive to the context, e.g. sex, age, location, and any other factor which affects ranges. May be used to represent high, low, normal, therapeutic, dangerous, critical, etc.
ranges that are constrained by an interval.
