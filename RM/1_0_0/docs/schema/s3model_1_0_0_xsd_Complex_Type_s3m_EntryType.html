Complex Type s3m:EntryType
Namespace http://www.s3model.com/ns/s3m/
Annotations
 
The abstract parent of all Entry subtypes. An Entry is the root of a logical set of data items. Each subtype has an identical information structure. The subtyping is used to allow persistence to separate the types of Entries; primarily important in
healthcare for the de-identification of clinical information.
