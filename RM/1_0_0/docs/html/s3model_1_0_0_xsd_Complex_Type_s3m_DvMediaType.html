Complex Type s3m:DvMediaType
Namespace http://www.s3model.com/
Annotations
 
A specialisation of DvEncapsulated for image, audio and other media types with a defined MIME type.
Includes further metadata relating to media types which are not applicable to other subtypes of DvEncapsulatedType.
